# Anthropic Course

## Setup

### First time setup

1. Create the virtual environment: `python3 -m venv .venv`
2. Activate it with: `source .venv/bin/activate`
3. Install dependencies: `pip install -r requirements.txt`
4. Create a `.env` file by copying `.env.example` and adding your actual API key

### Using the environment

Once set up, to use your environment in the future:

1. Activate it with: `source .venv/bin/activate`
2. You can now write your Jupyter notebooks or Python scripts using the Anthropic API
